Результатом работы программы будет 2 1. 
defer добавляет переданную после него функцию в стэк. 
При возврате внешней функции, вызываются все вызовы,
добавленные в стэк. Принцип работы стэка - LIFO (last in first out).
Если есть именованные возвращаемые параметры, отложенная функция может их читать и изменять.

func test() (x int): **x - именованный возвращаемый параметр, объявленная в обл-ти видимости всей ф-ии**
1. Именованной переменной 'x' присваивается значение 1
2. Перед возвратом из ф-ии выполняется отложенная ф-ия defer, увеличивая 'x' на 1
3. Возврат текущего значения x (=2)

func anotherTest() int: **возврат значения типа int**
1. Объявляется локальная переменная 'x'
2. Локальной переменной 'x' присваивается значение 1
3. Перед возвратом из ф-ии значение 'x' фиксируется
4. Выполнение отложенной ф-ии изменяет значение локальной переменной,
но никак не затрагивает изменения значения на п.3
5. Возврат значения x (=1)
