### 7
Результатом работы программы будут цифры 1-8 (порядок не гарантирован).
В ф-ии ```main``` с помощью ф-ии ```asChan``` мы создаем 2 канала на чтение: a,b

```func asChan(vs ...int) <-chan int```
- применяется стандартный паттерн генератор, который создает канал и возвращает его (на чтение)
- в ф-ии запускается горутина, которая через нефиксированные промежутки времени отправляет переданные данные в канал
- по окончании работы горутина закрывает канал

```func merge(a, b <-chan int) <-chan int```
- применяется стандартный паттерн генератор, который создает канал и возвращает его (на чтение)
- считывает из 2 каналов и записывет результат в 1
- работа ```select```. Select - блокирующий оператор. Мы ждем, пока разблокируется какой-то из case-ов. Если готов ```case v, ok := <-a:``` (то есть из канала a можно считывать данные), то обрабатываем этот случай. Мы не знаем, какой из каналов получит значение раньше, поэтому порядок цифр в итоговом ответе не гарантирован. При готовности 2-х case-ов одновременно выбирается случайный. Работа завершается в тот момент, когда оба канала будут = nil.

В ф-ии ```main``` создается канал на чтение c. Далее цикл ```range``` считывает данные из канала до тех пор, пока он не пуст и не закрыт, и выводим последовательно полученные значения.
